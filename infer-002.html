Comprehensive AI, Chatbot & Inference Services Dashboard

An extensive list of publicly accessible generative AI, chatbot services, inference platforms, and tools. Prioritizing free services or those with generous free tiers.
Projected as of: May 15, 2025. Note: The AI landscape is highly dynamic; features and availability may change.

Key for Tags:

Web: Accessible via a web browser. macOS: Native or well-supported macOS application. Windows: Native or well-supported Windows application. Linux: Native or well-supported Linux application. API: Offers an Application Programming Interface for developers. Free: Core service is free or has a substantial free offering. Generous Tier: Offers a significant free tier for exploration/light use.

Web-Based Chatbots & Conversational AI

Service Name Access Key Features / Free Tier Details Link
OpenAI - ChatGPT Web API Free Access to GPT-3.5 (unlimited), GPT-4o (rate-limited, text, vision, voice, data analysis, file uploads). Excellent reasoning, creativity. Context window up to 128k tokens for GPT-4 Turbo models (paid), GPT-4o offers strong capabilities on free tier. chat.openai.com
Google - Gemini Web API Free Access to Gemini 1.0 Pro. Gemini Advanced (paid) offers Gemini 1.5 Pro with 1M token context window. Strong multimodal capabilities, integration with Google Workspace. Free tier is very capable for general use. gemini.google.com
Anthropic - Claude Web API Generous Tier Claude 3 Sonnet/Haiku (free tier, rate-limited), Opus (paid). Known for safety, long context (up to 200K tokens), and strong reasoning. Free tier is excellent for text processing and summarization. claude.ai
Microsoft Copilot Web Windows Free Powered by OpenAI models (including GPT-4/GPT-4o at times) and Bing search. Image generation (DALL-E 3). Integrated into Windows, Edge, Microsoft 365 (paid for M365 integration). Free web version is powerful. copilot.microsoft.com
Perplexity AI Web API Generous Tier Conversational search with citations. Free tier uses their own model and provides good search-augmented answers. Pro version offers choice of models (GPT-4, Claude 3, etc.) and more "Pro" searches. perplexity.ai
Meta AI (Llama 3) Web Free Access to Llama 3 models. Integrated into Facebook, Instagram, WhatsApp, and a standalone website. Strong open model, image generation. meta.ai
Mistral AI - Le Chat Web API Generous Tier Access to Mistral Small/Large models (free tier likely rate-limited or uses smaller models). Known for efficiency and strong performance, especially open-weight models. chat.mistral.ai
Cohere - Coral / Playground Web API Generous Tier Enterprise-focused, but Playground offers free access for experimentation with Command models. Strong on RAG and tool use. Free tier for developers. cohere.com/playground
Hugging Face Chat Web Free Access to various open-source models (e.g., Llama, Mistral fine-tunes). Community-driven. Great for trying out new OS models. huggingface.co/chat
Poe by Quora Web macOS Generous Tier Access multiple models (ChatGPT, Claude, Llama, Gemini, etc.) in one interface. Free daily message credits for various bots. Subscription for higher limits. poe.com
xAI - Grok Web (via X Premium) Requires X Premium subscription. Known for real-time information access from X and a rebellious streak. x.ai/grok (Info)
DeepSeek Chat Web API Free Access to DeepSeek models, strong in coding and reasoning. Generous free API credits often available. chat.deepseek.com
Character.AI Web Free Create and chat with AI characters. Focus on personality and role-playing. Free tier is very usable. beta.character.ai
You.com (YouChat) Web Generous Tier AI-powered search engine with chat modes. Free tier for basic chat and search. Paid plans for advanced AI modes. you.com
Phind Web Generous Tier AI search engine for developers. Free tier provides useful answers for coding questions. Pair programming features. phind.com

Local Inference Platforms (Run AI Models on Your Own Machine)

Tool Name Platform Key Features / Free Tier Details Link
Ollama macOS Linux Windows Free Open-source. Easily run large language models locally (Llama, Mistral, Phi, etc.). CLI and API. Growing community support. Excellent for experimentation. ollama.com
LM Studio macOS Windows Linux Free User-friendly GUI to discover, download, and run LLMs locally. Supports various model formats (GGUF). Built-in chat interface and local server. lmstudio.ai
GPT4All macOS Windows Linux Free Open-source ecosystem to run powerful & customized LLMs locally. Chat client and model downloads. Privacy-focused. gpt4all.io
Jan macOS Windows Linux Free Open-source alternative to LM Studio. Aims to be an open and extensible local AI environment. Supports various models. jan.ai
Pinokio macOS Windows Linux Free Browser-like application to install, run, and control AI applications (including LLMs, image generators, etc.) locally with one click. pinokio.computer
