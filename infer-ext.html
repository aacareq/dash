General-Purpose Conversational AI Chatbots

User-friendly chatbots designed for a wide range of tasks like answering questions, writing, coding help, and general conversation.

Name (Provider) Capabilities Pricing Model Website
Microsoft Copilot (Microsoft) General conversation, web search integration (Bing), coding help, image generation, integrated into Windows/Edge/Office 365. Often uses latest OpenAI models (e.g., GPT-4). Generous Free Tier (often no sign-in for basic use), Paid versions via Microsoft 365. copilot.microsoft.com
Google Gemini (Google) Conversational AI, real-time web search integration, coding assistance, multimodal capabilities (text, image), integration with Google Workspace. Uses Gemini models (e.g., 1.5 Pro/Flash). Generous Free Tier (Gemini Pro/Flash), Paid (Gemini Advanced). Requires Google Account. gemini.google.com
ChatGPT (OpenAI) Industry standard, strong reasoning, writing, coding, math. Multimodal input/output (text, images, data analysis). Custom GPTs. Uses GPT-3.5, GPT-4o models. Robust Free Tier (GPT-3.5 / limited GPT-4o), Paid (Plus/Team plans for advanced models & features). Requires sign-up. chat.openai.com
Claude (Anthropic) Focus on safety, helpfulness, and detailed reasoning. Large context window for analyzing long documents. Strong coding & writing. Uses Claude 3 models (Haiku, Sonnet, Opus). Generous Free Tier (Sonnet/Haiku), Paid (Pro plan for Opus & higher limits). Requires sign-up. claude.ai
Perplexity AI (Perplexity AI) Conversational search engine focused on providing answers with real-time web citations. Good for research and fact-checking. Can use various models (incl. GPT-4, Claude). Good Free Tier (can often use without sign-up), Paid (Pro plan for more advanced model access & features). perplexity.ai
Grok (xAI) Real-time access to X (Twitter) data, witty tone, reasoning capabilities. DeepSearch mode. Uses Grok models. Paid (Requires X Premium+ subscription for full access), Limited free queries may be available. grok.x.ai (or via X platform)
Meta AI (Meta) General-purpose chatbot integrated into Meta platforms (WhatsApp, Instagram, Facebook). Uses Llama models (e.g., Llama 3). Image generation. Free (integrated into existing Meta services). meta.ai
Le Chat (Mistral AI) High-performance chatbot from Mistral AI, known for strong reasoning and multilingual capabilities. Uses Mistral Large/Small models. Free Chat Interface, Paid API access. Underlying models have open-weight options. chat.mistral.ai
Hugging Face Chat (Hugging Face) Platform allowing interaction with various open-source models (like Llama, Mistral variants). Customizable. Free (uses open-source models, usage limits may apply). huggingface.co/chat
Poe (Quora) Platform aggregating access to multiple AI models (ChatGPT, Claude, Gemini, Llama, etc.) in one interface. Allows creation of custom bots. Free Tier (limited access to various models), Paid subscription for higher limits and premium models. poe.com
YouChat (You.com) Search-integrated chatbot providing quick answers with citations, similar to Perplexity. Free Tier available. you.com/chat

LLM Inference Services (Developer Focused)

Platforms providing API access to large language models for developers to build custom applications.

Name (Provider) Capabilities Pricing Model Website
OpenAI API (OpenAI) API access to powerful models like GPT-4o, GPT-4, embeddings, image generation (DALLÂ·E), speech-to-text (Whisper). Fine-tuning options. Paid (Pay-as-you-go), Free trial credits often available for new users. platform.openai.com
Anthropic API (Anthropic) API access to Claude models (Haiku, Sonnet, Opus) known for safety, reasoning, and large context windows. Paid (Pay-as-you-go), Limited free tier/trial credits may be available upon request. www.anthropic.com/api
Google Cloud Vertex AI (Google) API access to Google's models (Gemini series, PaLM, Gemma). Integration with Google Cloud services. Multimodal capabilities. Fine-tuning. Paid (Pay-as-you-go), Free trial credits ($300) often available for new Google Cloud users. cloud.google.com/vertex-ai
Hugging Face Inference API (Hugging Face) API access to thousands of models hosted on Hugging Face Hub, including many open-source options (Llama, Mistral, etc.). Text, image, audio models. Free Tier (limited usage), Paid plans for higher throughput/dedicated infrastructure. Facilitates open-source model access. huggingface.co/inference-api
Mistral AI API (Mistral AI) API access to Mistral's high-performance models (Mistral Large, Small, 7B, Mixtral). Known for efficiency and open-weight options. Paid (Pay-as-you-go for API), Open-weight models available for self-hosting. mistral.ai/product/
Cohere API (Cohere) API access focused on enterprise NLP tasks like text generation, classification, embeddings. Strong multilingual support. Paid (Pay-as-you-go), Free tier for developers/experimentation. cohere.com/models
DeepSeek API (DeepSeek) API access to DeepSeek models, known for strong coding and reasoning capabilities. Some models are open-source. Free Tier (limited daily queries), Paid (Pay-as-you-go). platform.deepseek.com
GroqCloud (Groq) Ultra-fast LLM inference service using specialized LPU hardware. Supports open-source models like Llama, Mistral variants at very high speeds. Paid (Pay-as-you-go, often very competitive pricing), Free limited access may be available for experimentation. groq.com
Together AI API (Together AI) Platform providing fast inference for a wide range of open-source and some proprietary models. Focus on speed and cost-effectiveness. Paid (Pay-as-you-go), Free credits often available for new users. together.ai

Open-Source Models & Platforms

Models with publicly available weights or platforms primarily focused on hosting/running open-source AI.

Name (Provider/Origin) Capabilities Pricing Model Access / More Info
Llama Series (Meta AI) Highly capable family of models (Llama 2, Llama 3). Strong reasoning, coding, conversation. Different sizes available (e.g., 8B, 70B parameters). Open Source (License permits commercial use with restrictions). Free to download and run locally or via cloud platforms/APIs. ai.meta.com/llama
Mistral Open Weights (Mistral AI) Includes models like Mistral 7B and Mixtral 8x7B. Known for high efficiency and performance relative to size. Apache 2.0 license. Open Source (Apache 2.0). Free to download and use. mistral.ai/technology
Gemma Series (Google) Family of lightweight, state-of-the-art open models built from the research behind Gemini. Available in different sizes (e.g., 2B, 7B). Open Source (License permits commercial use). Free to download and use. ai.google.dev/gemma
DeepSeek Open Source Models Includes DeepSeek Coder and general LLM models. Known for strong performance, particularly in coding tasks. Open Source (License permits commercial use). Free to download and use. DeepSeek GitHub
Hugging Face Hub (Hugging Face) Central platform for discovering, downloading, and sharing thousands of open-source AI models (LLMs, vision, audio), datasets, and demos (Spaces). Platform is Free to browse/download. Paid tiers for private repos, accelerated inference, etc. huggingface.co/models

Niche & Specialized Tools

AI tools focused on specific applications like personal companionship, character interaction, or business tasks.

Name (Provider) Capabilities Pricing Model Website
Pi, your personal AI (Inflection AI) Designed to be a supportive and conversational companion. Focuses on empathetic conversation rather than task completion. Very easy access (Web, App, WhatsApp). Free. pi.ai
Character.AI (Character Technologies) Platform for creating and interacting with AI characters (fictional, historical, custom). Focus on role-playing and entertainment. Free Tier available, Optional paid subscription (c.ai+) for faster responses and early access. beta.character.ai
Replika (Luka, Inc.) AI companion designed for emotional connection and conversation. Offers features like journaling and coaching. Free Tier available, Paid Pro version unlocks more features and customization. replika.ai
Jasper (Jasper AI) AI writing assistant focused on marketing and business content creation. Offers templates, brand voice features, and integrations. Paid (Subscription plans), Free trial often available. jasper.ai
Tidio (Tidio) Customer service platform with AI chatbot features (Lyro AI). Focuses on automating customer support for small/medium businesses. Free Plan available (limited features), Paid plans for more capabilities. tidio.com
